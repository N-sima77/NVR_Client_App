When running this software(.exe), 
it is necessary that the PS-API dlls are put in "bin" directory.

- ipropsapi.dll
- ipropsapidevice.dll
- ipropsapivideo.dll
- ipropsapiftp.dll
- ipropsapishmlogger.dll


When you build this sample program, 
it is necessary that the PS-API header files are put in "include" directory.

- ipsapi.h
- iappcallback.h
- iapplistener.h
- isearchresult.h
- ipsapipicture.h
- ipsapifileinfo.h
- ipsapiftpfilelist.h
- psapidef.h
- ipropsapi.lib
